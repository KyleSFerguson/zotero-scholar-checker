Welcome to Zotero Scholar Checker!

To start seeing Zotero badges on Google Scholar, you need to connect this extension to your Zotero account.

Step 1: Get Your User ID

Go to your Zotero API Keys page. At the very top, look for the phrase: "Your userID for use in API calls is XXXXXXX".

Step 2: Create a Secret API Key

On that same settings page, click Create new private key.

  • Name it "Scholar Checker".
  • Make sure the Allow library access box is checked.
  • Click Save, and immediately copy the long secret key it gives you!
Your key is stored securely on your local browser and is never shared.
